Tic-Tac-Toe
=====================

Implementation from scratch of tic-tac-toe in a 3D world.  
__Frameworks:__ OpenGL (C++)

## Usage
```
$ git clone git@github.com:zydeon/tictactoe.git
$ cd tictactoe
$ make
```

### Keys

__X__: switch player  
__W__: ↑  
__A__: ←  
__S__: ↓  
__D__: ➝  
__Esc__: quit the game

For playing in the board:

| T  | Y | U |
:----:|:---:|:----:
| __G__ | __H__ | __J__ |
| __B__ | __N__ | __M__ |

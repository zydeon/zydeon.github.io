PeMo

Dec 1, 2013

An XMPP Instant Messaging client pontarius-xmpp using vty-ui as user interface. Includes support for Jabber.

Language: Haskell

Installation
$ cabal sandbox init
$ cabal install

  • Mac OSX: In order to install text-icu, you need the C libraries: icuuc, icui18n, icudata
    1. $ brew install icu4c
    2. $ cabal install text-icu --extra-include-dirs=/usr/local/Cellar/icu4c/53.1/include/ --extra-lib-dirs=/usr/local/Cellar/icu4c/53.1/lib

Or the equivalent using MacPorts.

Usage
$ cabal configure
$ cabal run

GL-TicTacToe

Jun 1, 2013

Implementation from scratch of Tic-Tac-Toe in a 3D environment.

Frameworks: OpenGL
Language: C++

Keys

X: switch player
W: ↑
A: ←
S: ↓
D: ➝
Esc: quit the game

For playing in the board:

T Y U
G H J
B N M

Installation

$ git clone git@github.com:zydeon/tictactoe.git
$ cd tictactoe
$ make
